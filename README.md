Для начало работы требуется скомпилировать файл Main.pas

Синие квадратики -- бактерии, берющие энергию от света
Красные квадратики -- бактерии-хищники. Поедатели всего, что двигается.
Белые квадратики -- мертвые бактерии
Фиолетовые квадратики -- бактерии-хпадальщики. Поедатели белых клеток
Серые квадратики  --- бактерии, берющие энергию от "тьмы"
Зеленые квадратики -- бактерии, мысасывающие энергию из соседей

Прим нажатие клавиши "W" можно переключатся между режимами "Энергия" и "Тип"
"Энергия" -- бактерии красяться(от черного к белому) в соответствие с количеством энергии. Чем темнее бактерия, тем меньше энергие у нее.
"Тип" -- бактерии красяться соответствие со своим типом
